camera axis views

select an axis:

  • viewFitFront(); // View front of scene, looking along -Z
  • viewFitLeft(); // View left of scene, looking along +X
  • viewFitBack(); // View back of scene, looking along +Z
  • viewFitRight(); // View right of scene, looking along -X
  • viewFitTop(); // View top of scene, looking down -Y
  • viewFitBottom(); // View bottom of scene, looking up +Y
