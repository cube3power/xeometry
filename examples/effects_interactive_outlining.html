outlining

See user guide: Outlining

// show outlines

  • showOutline(); // Outline everything
  • showOutline("saw"); // Outline saw
  • showOutline(["saw#3.27", "saw#3.93"]); // Outline two objects

// hide outlines

  • hideOutline(); // Remove outlines
  • hideOutline("saw"); // Remove outlines on saw objects
  • hideOutline(["saw#3.27", "saw#3.93"]); // Remove outline from two objects

// set outline color

  • setOutlineColor([0, 0, 0]); //
  • setOutlineColor([1, 0, 0]); //
  • setOutlineColor([1, 1, 0]); //
  • setOutlineColor([0, 1, 1]); //

// set outline thickness

  • setOutlineThickness(2); //
  • setOutlineThickness(4); //
  • setOutlineThickness(8); //
  • setOutlineThickness(16); //
  • setOutlineThickness(24); //
