clipping

See user guide: Clipping

creating clipping planes:

  • createClip("clip1", {pos:[-130,-40,0], dir:[0,0,-1]}); // Create first clipping plane
  • createClip("clip2", {pos:[-130,10,0], dir:[0,-1,-1]}); // Create second clipping plane

destroying clipping planes: (TODO: breaks when all are destroyed)

  • destroy(); // Destroy all clip planes
  • destroy("clip1"); // Destroy first plane
  • destroy("clip2") // Destroy second plane

enabling clipping planes:

  • enable(); // Enable all clipping planes
  • enable("clip1"); // Enable first plane
  • enable("clip2"); // Enable second plane

disabling clipping planes:

  • disable(); // Disable all clipping planes
  • disable("clip1"); // Disable first plane
  • disable("clip2"); // Disable second plane

showing clipping planes:

  • show("clip1"); // Show first plane
  • show("clip2"); // Show second plane

hiding clipping planes:

  • hide("clip1"); // Hide first plane
  • hide("clip2"); // Hide second plane

controlling which objects are clipped

  • setClippable() // Make scene clippable
  • setUnclippable() // Make scene unclippable
  • setClippable("saw") // Make saw clippable
  • setUnclippable("saw") // Make saw unclippable
  • setClippable(["saw#3.1", "saw#3.2"]) // Make cover unclippable
  • setUnclippable(["saw#3.1", "saw#3.2"]) // Make cover clippable
