opacity

set opacities of models and objects

  • setOpacity("saw", 0.2) // Set saw 20% opaque
  • setOpacity("saw", 0.4) // Set saw 40% opaque
  • setOpacity("saw", 0.6) // Set saw 60% opaque
  • setOpacity("saw", 1.0) // Set saw 100% opaque
  • setOpacity(["saw#3.1", "saw#3.2"], 0.2) // Set cover 20% opaque
  • setOpacity(["saw#3.1", "saw#3.2"], 0.4) // Set cover 40% opaque
  • setOpacity(["saw#3.1", "saw#3.2"], 0.6) // Set cover 60% opaque
  • setOpacity(["saw#3.1", "saw#3.2"], 1.0) // Set cover 100% opaque
