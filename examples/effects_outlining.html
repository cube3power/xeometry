outlining

show and hide outlines

  • showOutline()
  • showOutline("gearbox")
  • showOutline(["gearbox#1", "gearbox#1.0", ...])
  • hideOutline()
  • hideOutline("gearbox")
  • hideOutline(["gearbox#1", "gearbox#1.0", ...])

set outline color

  • setOutlineColor([1, 0, 0])
  • setOutlineColor([1, 1, 0])
  • setOutlineColor([0, 1, 1])
  • showOutline("gearbox")
  • showOutline(["gearbox#1", "gearbox#1.0", ...])

set outline thickness

  • setOutlineThickness(2)
  • setOutlineThickness(4)
  • setOutlineThickness(8)
  • setOutlineThickness(16)
